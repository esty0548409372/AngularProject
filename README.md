# Look In Book

### Generated with:
* [Angular CLI](https://github.com/angular/angular-cli) version 6.1.5. - frontend    
* node js - backend

### This project used :
* local-storage
* httpClient
* cors
* form validation

## Run this app in dev mode- client side:
```bash
clone 'https://github.com/NechamaShraiber/ProjectAngular'
cd BooksStore
npm i
ng serve --open
```
## Run this app in dev mode- server side:
```bash
clone 'https://github.com/NechamaShraiber/ProjectAngular'
cd ServerBookStore
npm i
npm start
```
